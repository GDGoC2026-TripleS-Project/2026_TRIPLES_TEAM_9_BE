# 2026_TRIPLES_TEAM_9_BE
대학 연합 프로젝트 TripleS BE 9팀 레파지토리입니다.

### Commit Convention
+ feat: 새로운 기능 추가
+ fix: 버그 수정
+ docs: 문서 수정
+ style: 코드 포맷팅, 세미콜론 누락 등 수정(코드 변경 없음)
+ refactor: 코드 리펙토링
+ test: 테스트 코드 작성
+ chore: 빌드 업무 수정, 패키지 매니저 수정

### issue 생성 후 커밋
+ #이슈번호 feat: 새로운 기능 추가
+ #이슈번호 fix: 버그 수정
+ #이슈번호 docs: 문서 수정
+ 등등

ex) git commit -m "#1 feat: 로그인 기능 구현"
